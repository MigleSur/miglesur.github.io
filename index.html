The list with links to interactive visualizations:

  1. Meteor showers visualization

    Good:Very intuitive and easy to interact with. The fact that the visualization reflects how the data looks like by mirroring or simulating reality, instead of resorting to abstract plots representations, makes grabbing its insights straightforward.

    Improvable:The view mode "Watch from Earth" is a bit confusing to the eye.

  2. The Gapminder project

    Good:Extremely comprehensive visualizations on several human development indicators for all countries in the world for more than a hundred years in some cases. The visualizations are also very intuitive to understand and fun to watch. The data displayed on them can be easily filtered. For example, one can select a specific set of countries

    Improvable:Contrary to the first visualization, it is very plot based and there are no visualizations where one can actually look at the data in such a direct way.
